# Netflix-python-project
# 🎬 Netflix Data Analysis using Python & Pandas

## 📌 Project Overview

This project focuses on analyzing the Netflix Movies and TV Shows dataset using **Python**, **Pandas**, and data visualization libraries. The goal is to perform **Data Cleaning**, **Exploratory Data Analysis (EDA)**, and extract meaningful business insights from real-world streaming platform data.

The dataset contains information about Netflix content, including titles, directors, cast members, countries, release years, ratings, durations, and genres. Through various analytical techniques, this project uncovers trends in content production, audience ratings, country-wise distribution, and platform growth.

---

## 🎯 Objectives

- Clean and preprocess raw Netflix data.
- Handle missing values and duplicate records.
- Perform Exploratory Data Analysis (EDA).
- Analyze content trends across years and countries.
- Explore ratings, genres, directors, and cast information.
- Generate actionable insights using data visualization.
- Apply real-world data analytics techniques using Python.

## 🔍 Business Questions Solved

### 1. Data Quality Analysis
- Identified and removed duplicate records.
- Detected missing values using heatmap visualization.

### 2. Content Exploration
- Retrieved details for specific titles such as **House of Cards**.
- Identified directors and show information.

### 3. Release Trend Analysis
- Determined the year with the highest number of releases.
- Visualized yearly content growth using bar charts.

### 4. Content Distribution
- Calculated the total number of Movies and TV Shows.
- Compared content categories through visualizations.

### 5. Country-wise Analysis
- Identified TV Shows released exclusively in India.
- Determined the country with the highest number of TV Shows.

### 6. Director Analysis
- Found the Top 10 directors with the most content on Netflix.

### 7. Genre-Based Analysis
- Filtered Movies categorized as Comedies.
- Analyzed Drama Movies and Kids TV content.

### 8. Cast Analysis
- Determined the number of titles featuring Tom Cruise.

### 9. Rating Analysis
- Identified unique Netflix ratings.
- Analyzed TV-14 Movies in Canada.
- Analyzed TV Shows with R ratings after 2018.

### 10. Duration Analysis
- Extracted and analyzed content duration.
- Found the longest content available on Netflix.

### 11. Sorting & Filtering
- Sorted content by release year.
- Applied advanced filtering using multiple conditions.
